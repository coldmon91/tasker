# Tasker 일정 관리 웹 앱

- Rust Tauri + SvelteKit 기반 웹 앱

## 기능
- Task 관리
- 리스트 뷰
- 캘린더 뷰 

## Task 관리
- Google Tasks, Apple Reminders 와 연동
- 할 일 추가, 수정, 삭제
- 우선순위 설정 (높음, 중간, 낮음)
- 마감일 설정
- 카테고리 설정
- 메모 추가
- 완료 표시
- 반복 설정 (매일, 매주, 매월, 매년)

## 캘린더 뷰
- 월간, 주간, 일간 뷰
- 드래그 앤 드롭으로 일정 이동
- 일정 색상 지정
- 반복 일정 설정
- 알림 설정
- 일정 검색 기능
- 일정 필터링 (카테고리, 우선순위 등)
- Google calendar 연동
